                                                                                  Gabriel Theodoropoulos Alicia V Carr a year ago

                                                                                  Hi,

                                                                                  Firstly, thank you very much!

                                                                                  Surely, you can establish a connection between two peers only, as long as none of them will invite other people to the session. Also, before a connection gets established, a confirmation alert view appears which you can use to decline other invitations made to you.

                                                                                  I'm not quite sure what exactly you mean with the location, so please explain more if you'd like.

                                                                                  Lastly, I guess that when you're saying about adding a camera, you mean to implement live chat. Well, what you actually want to do here is to perform streaming between the peers, and that's the third purpose of the multipeer connectivity framework, and also the one that was not covered in any of these two tutorials. If you manage to implement streaming, then combining it with the multipeer connectivity wouldn't be hard. Anyway, the final answer is yes, you can have live cam functionality.
