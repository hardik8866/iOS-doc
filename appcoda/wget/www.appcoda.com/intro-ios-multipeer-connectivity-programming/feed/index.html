I see you are doing in the main thread. I think it’s much better to make a queue and then call the dispatch_async.

dispatch_queue_t newQueue = dispatch_queue_create(“ProgressQueue”, NULL);
dipatch_async(newQueue, ^{
//Perform nsprogress operations
dispatch_async(dispatch_get_main_queue( ), ^{
//Update the UI
});
});
