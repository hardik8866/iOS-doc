The complete guide provides all the information you need to create your own game.

  • WHAT IS THE ADOBE ILLUSTRATOR FILE FOR?

    The AI file only comes with the complete package. It contains the original screen and character design of the iOS game. The design file makes it easier for you or your designer to change the game graphics and create your own game.

  • COULD I MODIFY THE SOURCE CODE?

    Absolutely! You are allowed to use or modify the source code included in the game kit. No attribution is required. But you’re not allowed to distribute or sell the source code without prior authorization.

  • WHAT VERSION OF XCODE AND IOS DO I NEED?

    You need to have Xcode v5.0 or up to use the game kit. The iOS game supports both iOS 6 and 7.

  • DO I NEED TO JOIN THE IOS DEVELOPER PROGRAM?

    You can still build and run the game using iPhone Simulator. However, without joining the iOS Developer Program, you will not be allowed to publish your app on App Store.

  • I GOT A GREAT APP IDEA. CAN THE IOS GAME KIT HELP?

    The game kit is not for you to make something big. The iOS Game Starter Kit is designed to help you jump start the app development in a small scale. You'll learn how to re-skin an app and the app publishing process. You'll also learn how to monetize from the app by integrating ads from different ad networks and selling power-ups via in-app purchase.
