]]> By: Special Comment http://www.appcoda.com/objective-c-blocks-tutorial/#comment-56531 Sun, 28 Sep 2014 16:43:00 +0000 http://www.appcoda.com/?p=2905#comment-56531 You cannot simply use ‘self’ inside the block as it creates a retain cycle.

The workaround is to create a __unsafe_unretained before using it in the block.

LIke this.

__unsafe_unretained typeof(self) weakSelf = self;

[self addNumber:1 withNumber:2 andCompletionHandler:^(int result) {

weakSelf.button.titleLable.text = [NSString stringWithFormat:@”Sum is %d”, result];

}];

Hope this helps
