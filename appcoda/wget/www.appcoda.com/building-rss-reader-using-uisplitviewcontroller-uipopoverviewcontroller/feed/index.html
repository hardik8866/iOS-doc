]]> By: Alexandra http://www.appcoda.com/building-rss-reader-using-uisplitviewcontroller-uipopoverviewcontroller/#comment-92375 Sun, 14 Dec 2014 09:33:00 +0000 http://www.appcoda.com/?p=4102#comment-92375 Thank you so much for this great tutorial, I learned a lot while following it!

I struggled with one deprecated delegate method, so I thought I’d post the solution I found:

In order to notify about display mode changes, the tutorial implements in the AppDelegate:

NSNotificationCenter.defaultCenter().postNotificationName(“PrimaryVCDisplayModeChangeNotification”, object: NSNumber.numberWithInteger(displayMode.toRaw()))

And in the TutorialViewController:
if nextDisplayMode == UISplitViewControllerDisplayMode.PrimaryHidden.toRaw()

But the toRaw method is now deprecated and replaced by rawValue – and the changes weren’t initially obvious to me, only found it via trial and error:
NSNotificationCenter.defaultCenter().postNotificationName(“PrimaryVCDisplayModeChangeNotification”, object: displayMode.rawValue)

if nextDisplayMode == UISplitViewControllerDisplayMode.PrimaryHidden.rawValue
