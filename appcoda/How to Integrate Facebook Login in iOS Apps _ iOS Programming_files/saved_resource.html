                          Gabriel Theodoropoulos Coeus a year ago

                          Hi,

                          The first FBLoginView delegate method that gets called once the login process is over (and successful), is the loginViewShowingLoggedInUser:. In there, you should perform the appropriate segue for loading another view controller, or do any other post-login actions. That's the half answer though. In case you want to use the returned user info, then you should do all actions in the loginViewFetchedUserInfo:user: delegate method, and probably you should pass the user object to the next view controller so you can work with it.
