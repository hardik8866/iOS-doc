                                    Hey not sure if you've found the solution, but I had the same issue so hopefully this helps anyone else with this problem. Make sure that whatever you name your identifier in the storyboard for the prototype cells matches the variable you add into the RecipeBookViewController.m. For example:

                                    static NSString *simpleTableIdentifier = @"RecipeCell"; //<-- this is case sensitive! I originally had it as @"recipeCell" which is different from the storyboard if you're following the exact naming as the tutorial.

                                    I made the R capital and problem solved for me :)
